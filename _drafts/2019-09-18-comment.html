Gotchas


  • In order to use the OpenRecord on a view you will need to make sure you have your records primary name field as the first clickable link in the view.  Depending on the entity I sometimes just create a "UI Testing" view which has this and change the view before the OpenRecord, then just set the view to be only viewable by certain security roles.
