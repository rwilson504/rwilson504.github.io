If you need to install the XPS viewer on Windows Server 2008R2 just follow these directions.  
  

1. Open the **Server Manager** for the computer.

2. Click on **Add features**.

[![](/images/install-xps-viewer-for-windows-server/01-XPS2.png)](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEhnogqiPGfslsMWbCSYTgKWKAxKp3sMGLtl_3-3iwV2cDVhLGCdIORUMimsiDO71Z2WmgSL0Kb9FnRtO3asCCNXBY4_zBff1B4I3Gk4oyCx-LkgqypHVv6bu_BsAju-TOvcaINZ0KCHXqY/s1600/XPS2.png)

3. Scroll down to the **XPS Viewer** selection and click the check box.

[![](/images/install-xps-viewer-for-windows-server/02-XPS3.png)](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEibo3D9lbuqp546jiQJ-sVrWv63BORgywzRs3DR5fidMPsgF6vyxBfIJSMOJ7xQFIFjVG-ly6lz_cQTuaVOrUEuTaejKSFRG-537L2Vti2bCAllvJYW6Umw_JzbVA1_kZvUmkA7Ey6HJQs/s1600/XPS3.png)

4. Click the **Install** button.

[![](/images/install-xps-viewer-for-windows-server/03-XPS4.png)](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEgmXGldTEnG4k3mRqktyEL605A4AfEdJf4DFncjoFP2XGrjftOA-Z7FcMRwjycNDO9eoRfkiN7aPtwRFDL7Pq8NC7HdjxAy67tacV47-_aAUAb5_hsEhXzYHsOdy5X9H6WCp-2qDx6z5wY/s1600/XPS4.png)

5. After the installation has completed click the **Close** button.

[![](/images/install-xps-viewer-for-windows-server/04-XPS5.png)](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEjYexNCi_uatlllCQOZhin82oMLypayH0Ns3UzMm_LAtcwfCjEhTSCEOzXx1-NqfeLKy8Hfoev39muKDzHlpbujwbDgxco3qHfEQC5CSU7QBSO2Zdls4Qil3XRnblq5mdmrHvK7at72JUM/s1600/XPS5.png)
